EMTag

Problem this project tries to solve

EMTs need as much information about an unconscious patient as they can get. EMTag allows EMTs to read a patient's pertinent health information from an RFID tag on their person using an android app.

Why was this important to me

This project interested me because it has the ability to save lives. Most mobile applications don't have the ability to have this impact.

My role in this project

My role for this project was the user interface and design of the application. Going into the project, I had no UX or android experience. This project was for a hackathon so I only had 24 hours to figure this out. I also designed the icon you see above.

My take aways

My biggest take away from this project was the importance of delegation. We had 24 hours to put this app together and none of us had any experience with android. Because of this, it was very important that we layout a game plan and split up the tasks evenly.

  • Client: AT&T Smart Tags Hackathon
  • Date: 2014-11-15
  • Service: User Interface Design
